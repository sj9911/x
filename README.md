# x.sunnyjoshi.com

Personal experiment lab — where design meets AI, code meets curiosity.

## What is this?

A space for shipping experiments at the intersection of design, artificial intelligence, and creative code. Each project starts as a question. Some become products. All of them teach me something new.

## Experiments

- **AI Agent Workflows** — Multi-agent orchestration patterns and autonomous collaboration
- **Generative Interfaces** — UI components that respond to AI context
- **Vibe Prototypes** — Rapid creative coding experiments and interactions
- **Prompt Engineering** — Frameworks and patterns for working with LLMs
- **Design Systems** — Tokens, components, and scalable visual languages

## Tech

- Static HTML/CSS/JS
- Glassmorphism UI with light/dark mode
- Hosted on Netlify
- Domain via Namecheap
